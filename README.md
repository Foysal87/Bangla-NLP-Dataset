# Bangla-NLP-Dataset
Bangla NLP dataset. This repository contains sbnltk datasets, which were used in  [Bangla nlp toolkit - sbnltk](https://github.com/Foysal87/sbnltk) .
Also , Existing Datasets are being listed here!
